# 🏏 Turf Booking System

A full-featured web application built using **Django** for managing and booking sports turfs.  
It provides a seamless experience for users to search, book, and pay for turf slots, while turf owners can manage turfs, bookings, payments, and view analytics.

---

## 🚀 Features

### 👤 **User Features**
- User registration and login  
- View available turfs
- Book turfs with UPI payment  
- View booking receipt  
- Manage user profile  
- Clean UI for smooth experience  

### 🏢 **Owner Features**
- Owner registration and login  
- Add, edit, update, delete turfs
- View total bookings  
- Dashboard analytics (turf count, bookings, revenue, etc.)  
- Manage owner profile with UPI ID  

### 💰 **Payment**
- UPI QR code generated automatically  
- Transaction ID verification  
- Booking confirmation & receipt
